Module: Comments

Service de persistance & cache des commentaires. - La base de données est un pool MySQL/MariaDB (driver : **mysql2/promise**). - Un cache **LRU** en mémoire (taille = 1000 entrées) est utilisé pour accélérer la lecture d’un commentaire individuel.
